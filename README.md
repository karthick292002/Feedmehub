# Feedmehub
FeedMe Hub - Personalized Content Discovery Platform.
A modern, responsive web application built with React, TypeScript, and Vite that aggregates and personalizes content feeds across multiple categories including news, entertainment, movies, music, and social media.

- Personalized content feeds based on user preferences
- Trending content discovery
- Advanced search functionality
- Favorites management with drag-and-drop reordering
- Dark/light theme support
- Responsive design with shadcn/ui components

Built with: React, TypeScript, Redux Toolkit, Tailwind CSS, shadcn/ui, Framer Motion
